# Normal Mode
`"$n"j,"$m"k,h,l`
`b` -- Backward word
`w` -- forward Word
`e` -- move End of word
Capital Case -- ignore punctuation


`0` -- move line top
`^` -- move line top(ignoring spaces)
`$` -- move line end

`gg` -- move top of a file(go go)
`G` -- move end of file
    * `ggVG`

**Paragraph**

`{`
`}`
