# Basic

* `v`
* `V`
* `Ctrl` + `v`


* `i` = inner / inside
* `a` = around / all


* `w` -- word
    * `aw` -- including space
* `p` -- paragraph

### Composita

* `viw` -- select a word
* `vip` -- select a paragraph
* `va(`, `vi(`,  `va"`, `vi"` -- select inside or around xxx
* `%` -- jump corresponding brancket
