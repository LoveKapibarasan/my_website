# Basic

* `v`
* `V`
* `Ctrl` + `v`
* `gv` = reselect

* `i` = inner / inside
* `a` = around / all


* `w` -- word
    * `aw` -- including space
* `p` -- paragraph

### Composita

* `viw` -- select a word
* `vip` -- select a paragraph
* `vax`, `vix`(e.g. `va(`, `vi(`,  `va"`, `vi"`) -- select inside or around xxx
    * w = word except for simbols
    * W = space
* `%` -- jump corresponding brancket
