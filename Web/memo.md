## Port

* **コンピュータ内の“通信の入口番号**
* 1つのPCでも Webサーバー・DB・SSH …複数のサービスを同時に動かせるように
「どの通信はどのサービスへ渡すか？」を見分けるための番号 

* 0〜65535

| 通信内容          | 使用されるポート |
| ------------- | -------- |
| Web (HTTP)    | 80       |
| Web (HTTPS)   | 443      |
| SSH           | 22       |
| PostgreSQL DB | 5432     |
| MySQL DB      | 3306     |
