# 7z 
```bash

7z "$command" -"$option" "$Archive" "$file"
```

**Command**
* x -- extract
* a -- archive
* extract on current directory
**Options**
* `-y` -- no confirm
* `-o"$path"` -- output path
* `-p"$password"`
