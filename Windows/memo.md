### Disk

* `C:`
    * `C:\Windows`: OS
    * `C:\Windows\System32`: OS core binary, system command, DLL
    * `C:\Program Files`: `/usr/bin`, 64bit
    * `C:\Program Files (x86)`: 32bit
    * `C:\Users<username>`: Profile, Home
        * `C:\Users<username>\AppData\Local`: `~/.config`
    * `C:\ProgramData`: `/etc`, `/var`
    * `C:$Recycle.Bin`: `~/.local/share/Trash/files`
