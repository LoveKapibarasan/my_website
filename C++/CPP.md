#  C++ basic manual

## List of compilers

* GCC/G++
* Clang


## How to compile


### GCC/G++

```cpp
g++ program.cpp -o program
```

### Clang
```
clang++ program.cpp -o program
```

## How to execute

```bash
./program
```
## Comment
* `//`, `/**/`
## Print
```cpp
#include <iostream> 
using namespace std; // omit std
cout << "Hello," << "World " << var << " !" << endl;
// cout = console output
// <<  = insert
// endl = \n + buffer flush
// \\n = \n
```
