* CE = community edition
* Open source
* Java

**Structure**
```
Database
 └─ database 
    | public
     ├─ Tables
     ├─ Foreign Tables（from other server）
     ├─ Schemas
     ├─ Views, materialized Views
     ├─ Indexes
     ├─ Event triggers
     ├─ Functions, Aggregate Functions
     └─ Extensions
            data, property, (ER) Diagram
```
