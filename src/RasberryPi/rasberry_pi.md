### SSH Connection
```bash
hostname -I 
ssh pi_or_user_name@<ip_address>
```
```bash
sudo raspi-config
```
`→ Interface Options → SSH → Enable`

* Should be in the same network.

* mozc `Ctrl`+ `<Space>`
